# WRF Forcings Subset/Regrid Tool

Assist with regridding and subsetting WRF forcings in NetCDF format to ParFlow forcings in PFB format.

## Requirements

* Python >= 3.5
* esmpy
* numpy
* xesmf
* xarray
* dask
* pandas
* tqdm
* dask-jobqueue

## Usage

See example `wrf_to_pf_subset2.bash` file.
